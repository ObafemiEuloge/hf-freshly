Mixitup

Le mixitup est une bibliothèque JavaScript populaire utilisée pour créer des filtrages et des animations de tri pour les d'une page web. Il offre une solution flexible pour organiser, trier, et filtrer des contenus tels que des images, des vidéos, des articles de blog, des produits, etc.
