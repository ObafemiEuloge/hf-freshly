Bootstrap

Bootstrap est un framework de développement wrn open source très populaire. Il fournit des outils et des composants prédéfinis pour faciliter la création de sites web réactifs et esthétiquement agréables. Bootstrap utilise principalement du HTML, CSS, et JS pour la conception de l'interface utilisateur (UI)

CDN (Content Delivery Network) est un réseau de serveurs répartis géographiquement à travers le monde.
