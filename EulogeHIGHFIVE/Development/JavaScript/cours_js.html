Javascript

Javascript est un langage de programmation web créé en 1995 par Brendan Eich

ES est l'abbréviation de l'ECMAScript pour définir la version de Javascript

JavaScript est un langage côté client qui est interptété par le navigateur.

Les variables en JavaScript

Les variables sont des emplacements nommées pour conserver une valeur. Elles sont constituées d'un type, d'un nom et d'une valeur qu'on lui affecte.

  • string
  • entier

Les règles à respecter

  • Un bon principe de nommage des variables ;
    • La CamelCase pour nommer les variables
      Exemple : nomDeLaVariable = valeur
    • La snake_case pour nommer les variables
      Exemple : nom_De_La_Variable = valeur

A savoir en JS

const nom = valeur =============================> pour déclarer une variable constante;
alert("message") ===============================> pour afficher un message d'alerte dans le navigateur;
console("message") =============================> pour afficher un message ou une variable dans la console du navigateur;
document.write("message") ======================> pour afficher un message ou une variable sur la page;
let (ou var) maVariable = valeur ================> pour déclarer une variable pouvant prendre plusieurs valeurs ;
** ==============================================> représente l'opétateur d'exposant ;
new Date() ======================================> est un objet JavaScript qui permet de déclarer une date ;
getFullYear =====================================> récupère juste l'année d'une date ;
getMonth ========================================> récupère juste le mois de l'année d'une date ;
getDate =========================================> récupère juste le jour du mois [1 - 31] ;
En déclarant un tableau getDate ==========================================> récupère juste la second d'une date [0 - 59] ;
getDay ==========================================> récupère juste le jour de la semaine [1 - 2] ;
getSeconds ==========================================> récupère juste la second d'une date [0 - 59] ;
getHours ==========================================> récupère juste l'heure d'une date [0 - 23] ;
getMinutes ==========================================> récupère juste la minute d'une date [0 - 59] ;
getSeconds ==========================================> récupère juste la second d'une date [0 - 59] ;

Une concaténation est l'addition de deux ou plusieurs variables dont l'une est au moins une chaine de caratère.
